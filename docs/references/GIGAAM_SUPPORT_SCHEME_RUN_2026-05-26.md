## Техническое Решение

GigaAM загружался из локальных исходников `%USERPROFILE%\GigaAM`.

Для обхода проблемы `sentencepiece` с кириллическим путем к tokenizer использован ASCII-cache:

```text
%ProgramData%\gigaam_cache
```

В Git добавлен reusable wrapper:

```text

scripts/gigaam_transcribe_chunks.py
docs/operations/GIGAAM_TRANSCRIPTION.md
```
