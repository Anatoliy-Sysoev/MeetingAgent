# Контрольные Вопросы Для RAG

Обновлено: 2026-05-07.

Этот файл - стартовый набор для `FTT-MA-07`. Он нужен, чтобы измерять качество поиска до разработки карточек встреч, классификации и генерации документов.

## Как Заполнять

Для каждого вопроса фиксируем:

- вопрос;
- ожидаемые документы или типы документов;
- фактические top-k источники;
- замечания;
- решение: проходит / требует улучшения / провал.

Команда для проверки:

```powershell
cd "$env:USERPROFILE\Desktop\AI\MeetingAgent"
.\.venv\Scripts\python.exe scripts\04_query.py "Вопрос" --top-k 8 --compact
```

Для машинной фиксации результата:

```powershell
.\.venv\Scripts\python.exe scripts\04_query.py "Вопрос" --top-k 8 --raw
```
